# Pixabay — Image Search App

An Angular application that integrates the **Pixabay API** to search and browse high-quality free images.

## Tech Stack

| Layer | Technologies |
|-------|-------------|
| Frontend | Angular, TypeScript |
| API | Pixabay REST API |
| Styling | Bootstrap, CSS3 |

## Features

- Search images by keyword
- Masonry/grid image gallery
- Image detail view with metadata
- Pagination and lazy loading
- Responsive design
